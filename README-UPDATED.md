## 🚀 Getting Started

### Prerequisites
- Node.js 14+ (optional, for offline development)
- Modern web browser

### Installation & Running

**Option 1: Using npm serve (Recommended)**
```bash
npm install
npm run dev
```
Server starts at: `http://localhost:8000`

**Option 2: Using Node.js Server**
```bash
node server.js
```

**Option 3: Direct File Access**
Open `public/index.html` in your browser
